# Sorteio
Software simples, em Desktop - C#, para realizar sorteios.
Acrescente uma imagem para personalizar no pictureBox1.

Carregue arquivos de planilhas do Excel.
Selecione uma coluna com números inteiros, somente. Então rode o sorteio.
O valor da coluna logo à direita da coluna escolhida (ID) será apresentada como ganhador(a).
